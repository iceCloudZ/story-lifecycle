- 既有契约测试的 import 路径 `story_lifecycle.adapters.base` 已失效(BaseAdapter 搬到了 `story_lifecycle.knowledge.adapters.base`),本次一并修复。

- lifecycle 写锚点确实是纯文件操作,不 import miner,符合「卸包照跑」的 SOFT 设计。
