## 做了什么(What)

新建 `packages/story-lifecycle/tests/test_finding_lifecycle.py`,端到端验证 Finding 全生命周期:

1. `record_finding(...)` → DB 记录存在,`status == "open"`。
2. `update_finding_status(..., "accepted")` → 状态变为 accepted,并写入 `finding_status_changed` audit event。
3. 依次 `fixed` → `verified`,状态正确迁移。
4. `propose_learned_pattern(...)` → 产出 pattern,`status == "proposed"`。
5. `approve_pattern(...)` → `activate_pattern(...)` → `status == "active"`。

测试使用 `conftest.py` 的 `_isolated_db` autouse fixture,DB 指向 per-test `tmp_path`,不污染真实库。
