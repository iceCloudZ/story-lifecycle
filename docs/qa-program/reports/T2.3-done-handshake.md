## 做了什么(What)

新建 `packages/story-lifecycle/tests/test_done_handshake.py`,验证 `continue_orchestrator_agent`
对 `.story/done/<key>/<stage>.json` 的轮询握手:

1. **成功路径**:预置 done file,调用 `continue_orchestrator_agent`,断言 story 状态变为 `completed`,

2. **超时路径**:不创建 done file,调用 `continue_orchestrator_agent`,mock `time.sleep` 加速轮询,
   断言 story 状态变为 `failed` 且 `last_error` 含 "timed out"(证明不会无限挂起)。

测试使用 `headless-smoke` profile 走 headless 路径;`subprocess.Popen`、stdout supervisor、
`_kill_headless` 均被 mock,不启动真实 CLI。未修改 `pty.py`。
