## 做了什么(What)

新建 `packages/story-lifecycle/tests/test_gate_branches.py`,对 `run_verify_gate` 的三个判定分支各写了一个正向测试:

1. **advance**:无 HIGH findings → `decision == "advance"`。
2. **retry**:有 HIGH findings 且当前 round 低于 `max_retries` → `decision == "retry"`,返回的 reason 含 round 计数。
3. **fail**:有 HIGH findings 且 round 超过 `max_retries` → `decision == "fail"`。

测试中 mock 了 `judge_verify_stage`、`get_open_findings`、`log_event` 和 `build_repair_packet`,只测判定逻辑,不依赖真实 LLM 或 packet 构造细节。
